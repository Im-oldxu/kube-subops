## 8. 当前问题与风险

- 风险 1：首版范围覆盖全资源、全表单和全高危操作，后续实现必须严格按阶段拆分。
- 风险 2：SQLite 适合单实例快速版，不适合多实例和高并发审计增长。
- 风险 3：Sub Admin 模板基线已复制并通过 `typecheck` / `build`，但尚未保存为 Git 模板基线；进入并行开发前应先提交该基线。
- 风险 4：正式 Kubernetes 导航、业务页面、service/mock 和后端联调尚未开始。
